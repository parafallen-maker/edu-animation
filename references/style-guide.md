# 视觉风格指南

Manim 教育视频的视觉标准，受 Kurzgesagt/回形针风格启发。

## 核心美学

- **扁平设计**：最少阴影，干净几何形状
- **暗色背景**：深蓝基调，高对比度前景
- **几何形状**：圆角矩形、圆形、三角形的组合
- **色彩丰富但不杂乱**：每场景最多 2-3 个强调色
- **一致性**：同一元素全程使用相同颜色

## 颜色系统

所有颜色必须从 `design_system.py` 的 `T` 类导入，禁止硬编码。

### 背景色
```python
T.BG         = "#0B1120"  # 主背景 — 深空蓝
T.BG_MEDIUM  = "#132040"  # 次级背景 — 中蓝
T.BG_LIGHT   = "#1E3055"  # 高亮背景 — 亮蓝
```

### 强调色
```python
T.BLUE    = "#4A9EFF"  # 信息、数据
T.GREEN   = "#4ADE80"  # 成功、增长
T.RED     = "#FF6B6B"  # 警告、危险
T.YELLOW  = "#FBBF24"  # 高亮、重点
T.PURPLE  = "#A78BFA"  # 高级、特殊
T.ORANGE  = "#F97316"  # 强调、能量
T.TEAL    = "#00B8A9"  # 冷调、次要
T.ROSE    = "#E94560"  # 温暖、重要
```

### 语义色
```python
T.POSITIVE = "#00B894"  # 正确、好
T.NEGATIVE = "#E17055"  # 错误、差
T.NEUTRAL  = "#74B9FF"  # 中性信息
```

### 颜色使用规则
1. 背景始终使用暗色
2. 文字使用白色或 `T.BODY` / `T.MUTED`
3. 强调色 sparingly 使用，每场景 ≤3 个
4. 同一元素全程保持同一颜色
5. `rgba()` 半透明色可用于阴影/遮罩（豁免调色板检查）

## 字体系统

### 默认字体
```python
T.FONT = "PingFang SC"  # macOS 内置，中文完美支持
```

### 字号分级（1080p 画布）
```python
T.H1     = 56  # 主标题 — 用作默认，不是 36
T.H2     = 36  # 章节标题
T.H3     = 24  # 小标题、标签
T.H4     = 20  # 辅助文字
T.BODY_S = 16  # 正文
T.CAP    = 13  # 绝对最小值 — 任何字号不得低于此值
```

### 字号使用规则
| 位置 | 最低值 | 推荐值 |
|------|--------|--------|
| 场景主标题 | 36 | 56 (T.H1) |
| 章节标题 | 24 | 36 (T.H2) |
| 正文/标签 | 13 | 16-24 |
| 字幕 | 13 | 24 (T.H3) |
| **绝对最低** | **13** | — |

### 字号 ❌/✓ 示例
```
❌ Text("标题", font_size=10)  # 太小，低于绝对最低值
❌ Text("标题", font_size=48)  # 非标准值
✓  Z("标题", font_size=T.H1)  # 使用设计 token
✓  Z("正文", font_size=T.BODY_S)
```

## 形状语言

### 基础形状
```
┌─────────────────────────────────────────┐
│  Circle:        角色、图标、强调点       │
│  RoundedRectangle: 容器、卡片、面板     │
│  Arrow:         方向、流程、关系         │
│  Line:          连接、分隔、辅助线       │
│  AnnularSector: 饼图、环形图            │
│  Polygon:       星形、六边形装饰         │
└─────────────────────────────────────────┘
```

### 圆角标准（corner_radius）
```python
T.RADIUS = [0.05, 0.1, 0.2, 0.3, 0.5]
# 0.05 = 微圆角（按钮）
# 0.10 = 小圆角（小卡片）
# 0.20 = 中圆角（标准卡片）
# 0.30 = 大圆角（大面板）
# 0.50 = 圆形/胶囊
```

### 描边标准（stroke_width）
```python
T.STROKE = [1, 2, 3, 4]
# 1 = 细线（辅助）
# 2 = 标准线（边框）
# 3 = 粗线（强调）
# 4 = 超粗线（箭头、标注）
```

## 场景级调色板变化

可以在保持主调色板一致的前提下，为不同场景使用微调的背景色：

```python
# 在 Scene.setup() 中微调背景
def setup(self):
    self.camera.background_color = T.BG       # 默认深空蓝
    # 或使用微调变体
    # self.camera.background_color = "#0D1525"  # 更深
    # self.camera.background_color = "#0F1830"  # 稍亮
```

## 环境效果

Manim 中可用的环境增强技巧：
- 背景装饰：低透明度圆形/网格作为背景纹理
- 光晕效果：`GlowDot` 或自定义带模糊的圆
- 粒子效果：使用 `VGroup` + `randrange` 创建随机散布点
